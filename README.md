# Golang Data Structure

### 说明

1. 使用数组实现栈、队列、链表等基本数据结构，优点是执行效率高，缺点是提前确定最大容量后不能扩容。
2. 这些模板更适合参加算法竞赛，但也试用结构体和接口进行封装，体现了面向对象的思想
